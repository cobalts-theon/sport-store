## About The Project

**Prime Souls** is a comprehensive e-commerce platform designed for sports enthusiasts. Built with modern web technologies, it offers a seamless shopping experience with real-time search, advanced filtering, secure authentication, and a powerful admin dashboard.

### Key Highlights

- **Modern UI/UX** - Sleek dark theme with neon accents and smooth animations
- **Lightning Fast** - Built with Vite for optimal performance
- **Secure** - JWT authentication with Google OAuth integration
- **Fully Responsive** - Optimized for all device sizes

## Getting Started

### Prerequisites
- Node.js >= 18.x
- MySQL >= 8.0

### Quick Start

```bash
# Clone repo
git clone https://github.com/cobalts-theon/sport-store.git
cd Prime-Souls

# Backend
cd backend
npm install
# Configure .env (DB_HOST, DB_USER, DB_PASSWORD, DB_NAME, JWT_SECRET, EMAIL_USER, EMAIL_PASS)
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

**Access:** Frontend `http://localhost:5173` | API `http://localhost:3000`
